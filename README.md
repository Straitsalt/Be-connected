# Be-connected

A modern social media platform built with the MERN Stack.

## Features

- Google Authentication
- User Profiles
- Friend Requests
- Private Chat
- Group Chat
- Voice Messages
- Audio & Video Calls
- Posts & Comments
- Stories
- Image & Video Upload
- Large File Sharing
- Notifications
- Dark Mode
- Responsive Design

## Tech Stack

Frontend
- Next.js
- React
- Tailwind CSS

Backend
- Node.js
- Express.js

Database
- MongoDB Atlas

Authentication
- Firebase Authentication (Google Sign In)

Realtime
- Socket.IO

Storage
- Firebase Storage

Calls
- WebRTC
